# The Flavour Hub ☕

A modern, multi‑page café/restaurant website built with **HTML, CSS, and JavaScript**.  
This project showcases creative branding, responsive design, and clean UI structure — designed to be recruiter‑ready and easily deployable on GitHub Pages or Netlify.


 Features

 **Custom Branding**  
  Unique logo and vibrant color palette for a lively café identity.

 **Responsive Navigation**  
  Logo aligned left, navigation links aligned right, with mobile‑friendly stacking.

 **Multi‑Page Layout**  
  - `index.html` → Homepage with hero banner and welcome message  
  - `menu.html` → Expanded menu (coffee, starters, pizza, burgers, rolls, juices, desserts)  
  - `about.html` → Story and background of The Flavour Hub  
  - `contact.html` → Contact form and location details

 **Interactive UI Elements**  
  Buttons, hover effects, and subtle transitions for a polished user experience.

 **Ambience Gallery**  
  Showcases café interiors and atmosphere with responsive image grid.

 **Sticky Header**  
  Navigation bar stays visible while scrolling.

---

Tech Stack

- **HTML5** → Semantic structure  
- **CSS3** → Flexbox, responsive design, hover effects, shadows  
- **JavaScript** → Interactive elements (hamburger menu, form handling)  
- **GitHub Pages / Netlify** → Deployment and hosting


Deployment

### GitHub Pages
1. Push all files to the `main` branch.
2. Go to **Settings → Pages**.
3. Set **Branch** = `main` and **Folder** = `/root`.
4. Save — your site will be live at:  
   `https://<your-username>.github.io/<repo-name>/`
